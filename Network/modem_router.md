# Modem
is a device can convert internet cable (Fiber) from ISP to signal that our device can use.


![](/assets/modem.jpg)

# Router
is a device can distribute internet to multi device

Features:
- Wifi signal
- Gen IP (DHCP) and assign to device
- NAT
- Firewall

![](/assets/modem.jpg)

Modern Modem and Router is be in the same device



Device from ISP is be both of Modem and Router = gateway

![](/assets/internet-isp-router.png)
